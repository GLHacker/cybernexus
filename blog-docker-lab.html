Setting up a Home Lab with Docker

Posted on Jan 10, 2025 by Admin

Building a home lab is the best way to learn cybersecurity. Docker makes it lightweight, portable, and safe. Here is how to get started.

Why Docker?

Unlike heavy Virtual Machines (VMs), Docker containers share the host OS kernel, making them incredibly fast and efficient. You can spin up a vulnerable machine in seconds.

Step 1: Install Docker

On Ubuntu/Debian:

sudo apt update
sudo apt install docker.io docker-compose

Step 2: Pull Vulnerable Containers

Don't hack real sites! Use these intentionally vulnerable apps:

DVWA (Damn Vulnerable Web App)

docker pull vulnerables/web-dvwa
docker run -rm -it -p 80:80 vulnerables/web-dvwa

Juice Shop (OWASP)

docker pull bkimminich/juice-shop
docker run -rm -p 3000:3000 bkimminich/juice-shop

Step 3: Network Isolation

Create a secluded network for your lab so your main network stays safe.

docker network create --driver bridge isolated_lab

Conclusion

You now have a safe environment to practice SQL Injection, XSS, and more. Happy hacking!
